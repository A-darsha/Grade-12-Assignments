Setting Up XAMPP on Your PC

Home Assignments Download Install Configure Test

Download XAMPP

1. Visit the official XAMPP website: Apache Friends.

2. Choose the XAMPP version for your operating system (Windows, Linux, or OS X).

3. Click on the "Download" button to get the installer.

Install XAMPP

  1. Run the downloaded installer.
  2. Follow the on-screen instructions to complete the installation.
  3. Select the components you want to install (e.g., Apache, MySQL, PHP, etc.).
  4. Choose the installation directory (default is usually fine).
  5. Click "Next" and then "Install" to begin the installation process.

Configure XAMPP

1. Launch XAMPP Control Panel from the Start menu or desktop shortcut.

2. Start the Apache and MySQL services by clicking on the "Start" buttons next to each service.

3. Configure Apache and MySQL settings if needed. You can access the configuration files via the XAMPP Control Panel:

Apache Configuration: Config > Apache (httpd.conf)
MySQL Configuration: Config > MySQL (my.ini)

4. Open your web browser and navigate to http://localhost to check if Apache is running.

5. To manage MySQL databases, navigate to http://localhost/phpmyadmin.

Test Your Setup

1. Create a test PHP file to ensure PHP is working:

<?php
echo "Hello, XAMPP!";
?>

2. Save this file as test.php in the htdocs directory inside your XAMPP installation directory (e.g., C:\xampp\htdocs\).

3. Open your web browser and navigate to http://localhost/test.php. You should see the message "Hello, XAMPP!" displayed.
