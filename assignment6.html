INSERT Command

The INSERT command is used to add new rows of data into a table.

Example

INSERT INTO Employees (EmployeeID, FirstName, LastName, Department)
VALUES (1, 'John', 'Doe', 'HR');

Output:

EmployeeID FirstName LastName Department
1 John Doe HR

UPDATE Command

The UPDATE command is used to modify existing rows in a table.

Example

UPDATE Employees
SET Department = 'Finance'
WHERE EmployeeID = 1;

Output:

EmployeeID FirstName LastName Department
1 John Doe Finance

DELETE Command

The DELETE command is used to remove rows from a table.

Example

DELETE FROM Employees
WHERE EmployeeID = 1;

Output:

EmployeeID FirstName LastName Department
No rows No rows No rows No rows

SELECT Command

The SELECT command is used to retrieve data from a table.

Example

SELECT * FROM Employees;

Output:

EmployeeID FirstName LastName Department
1 John Doe Finance
